# Phoenix-AI-Interview-Platform
AI-powered Technical Interview Platform using Google Gemini, Streamlit, Speech Recognition, Live Coding, Proctoring and Automated Candidate Evaluation.


🔥 Phoenix AI Interview Platform

Conduct production-ready AI interviews with adaptive questioning, live coding assessment, voice interaction, automated evaluation, and intelligent hiring recommendations.

Overview

Phoenix is an enterprise-grade AI Interview Platform designed to automate technical hiring while maintaining a natural candidate experience.

Instead of asking static interview questions, Phoenix dynamically adjusts the interview based on:

Candidate Resume
Job Description
Experience Level
Previous Answers
Coding Performance

The platform provides recruiters with an explainable hiring recommendation supported by detailed analytics and scoring.
